QUICK START:

To use the feed system:

1. Review the API Documentation:
   cat /d/Project/docs/COOCOOK_FEED_API.md

2. Check the implementation:
   cat /d/Project/backend/services/feed_service.py

3. View the routes:
   grep -A 20 "SOCIAL FEED ENDPOINTS" /d/Project/backend/services/coocook.py

4. Run tests:
   pytest /d/Project/tests/test_feed_service.py -v

DATABASE TABLES:

- feeds (existing, used for feed generation)
- recipe_reviews (updated with helpful_count, unhelpful_count)
- recipe_likes (existing, used for like tracking)
- user_follows (existing, used for follow relationships)
- recipes (updated with title, share_count, update_rating method)
- users (existing, linked to all social entities)
- cooking_sessions (new, tracks user cooking sessions)
