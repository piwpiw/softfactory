┌─ OPTIMIZATION QUICK TIPS ──────────────────────────────────────────────────────┐
│                                                                                 │
│ If p95 > 500ms:                                                                 │
│ 1. Run Redis caching: pip install redis                                        │
│ 2. Add @cached(timeout=300) decorator to slow endpoints                        │
│ 3. Add database indexes (see PERFORMANCE_ANALYSIS.md Appendix C)                │
│                                                                                 │
│ If error rate > 1%:                                                             │
│ 1. Check Flask logs: tail -f backend/logs/error.log                            │
│ 2. Verify database connectivity                                                │
│ 3. Reduce VU count in load-test.js if using SQLite                             │
│                                                                                 │
│ If CPU > 80%:                                                                   │
│ 1. This is expected at 100+ VUs                                                │
│ 2. For production, use multiple Flask instances behind nginx                    │
│ 3. See: PERFORMANCE_ANALYSIS.md Section 6 "Scaling Guidelines"                │
│                                                                                 │
