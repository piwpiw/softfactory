About This API

SoftFactory is a comprehensive multi-service platform providing 5 specialized services: CooCook (Chef Booking), SNS Auto (Social Media Automation), Review (Influencer Campaigns), AI Automation (Business Automation), and WebApp Builder (Educational Bootcamp).

This API documentation includes 47+ endpoints across 10+ service modules, all protected by JWT authentication with optional demo mode support.

Authentication
