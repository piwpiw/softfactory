📱 SNS 자동화

SNS 자동화 API

GET /api/v1/sns/posts

연동된 SNS 게시물 목록 조회

{
  "posts": [
    {
      "id": "post_1",
      "content": "Hello World",
      "platforms": ["twitter", "facebook"],
      "status": "published",
      "created_at": "2026-02-24T10:30:00Z"
    }
  ]
}

POST /api/v1/sns/posts

새 게시물 등록

Request:
{
  "content": "New post",
  "platforms": ["twitter"],
  "scheduled_at": "2026-02-25T10:00:00Z"
}

Response: 201 Created
{
  "id": "post_2",
  "status": "scheduled"
}
