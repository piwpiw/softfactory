배포 가이드

현재 배포 기준은 정적 `web/` 출력과 `api/` 프록시를 Vercel에 올리는 방식입니다. 운영 체크 포인트는 배포 전 구조 감사, 배포 후 주요 경로(`/`, `/platform`, `/coocook`, `/analytics`) 응답 확인입니다.

1. 구조 확인

루트 오염, deprecated 경로, 누락 링크가 없는지 먼저 검사합니다.

2. 배포 실행

Vercel 프로덕션 배포로 정적 출력과 API rewrite를 함께 반영합니다.

3. 동선 점검

메인 진입 경로와 핵심 서브페이지 이동 중 404가 없는지 확인합니다.
